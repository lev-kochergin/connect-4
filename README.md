# Connect 4

## Start-Up
Run `connect.py` with your favorite tool, such as executing it with command prompt using:
```cmd
python3 connect.py
```

The project doesn't have any external depencencies, and was written in Python 3.9 and should be compatible with any other Python 3 release.
If you do not have Python installed, download the latest release first from `https://www.python.org/downloads/`
